CodeChallenge

It renders the code editor for the user, along with the buttons for testing and submitting the code.

new CodeChallenge(props: Object)

Extends React.Component

Parameters
props (Object)
Name Description
props.languages array The languages available for the challenge.
props.timelimit Number Time limit for the challenge's completion, in milliseconds.
props.id String Id to be used for the CodeEditor component.
props.onCodeTest Function Callback function for when the user sends his code for testing. It sends back the current editor's content.
props.onResponse Function Callback function for when the compiling API sends back a response. It sends back the API response.
props.onError Function Callback function for when the compiling API sends back an error or is unreachable. It sends back the error object.
props.onSubmit Function Callback function for when the user submits his code. It sends back the current editor's content.
Instance Members
▸ componentDidMount()

When the component mounts, it starts a timer with the challenge time limit, and upon reaching it, the editor's current content is automatically submitted.

componentDidMount()
▸ testCode

It sends the current editor's content to the compiler API for testing. It then reads the response and deal with the necessary callback function. If the API responds with an error property or is unreachable, and error is thrown. the editor's current content is automatically submitted. Please note that this ideally shouldn't be hardcoded here.
